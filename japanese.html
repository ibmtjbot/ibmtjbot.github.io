彼に命を吹き込むために、あなたが追加できる部品が沢山あります。
これら全てが全ての開発例で必要というわけではありません。

Raspberry Pi 3 + SD card preloaded with NOOBS.
This is a required component to make TJBot work! 🤖
USB Microphone.
Other brands of USB microphones should also work.
Mini Bluetooth Speaker.
Any small speaker with either a 3.5mm audio jack or Bluetooth will work. Note that if you are using the 3.5mm audio jack, you may wish to add a USB Audio Adapter to avoid audio interference with the LED.
Servo Motor.
Note that the red (middle) wire is 5v, the brown wire is ground, and the orange wire is data.
NeoPixel RGB LED (8mm).
If you are using other kinds of LEDs, you may need to add a resistor; this LED doesn’t require one.
Female-to-female + female-to-male jumper wires.
TJBot will only need 3 of each type, so you’ll have extra.
Raspberry Pi Camera. Either the 5MP or 8MP camera will work.

TJBotに命を吹き込む！

開発例はTJBotをSpeech to Text、Visual Recognition、Language TranslatorのようなWatsonサービスとつなぐのに役立つステップバイステップの解説です。 開発例は、Raspberry Pi用に設計されています。 あなたは、利用可能な開発例の1つを動かすか、自分で開発することができます。

ぜひ、githubやInstructablesで開発例を見てみてください。

あなたがTJBotで開発したものを見るのが待ち切れません！

IBM Watson Maker Kitsは、楽しくWatsonサービスに接続するためのDIYオープンソーステンプレート集です。 TJBotは、コレクション内の最初のメーカーキットであり、コグニティブオブジェクトの設計と実装におけるベストプラクティスを見つけるための実験として、IBM ResearchのMaryam Ashooriによって作成されました。 TJBotは、このブログ投稿を介して2016年11月9日に生まれました。彼の名前は、親しみを込めてIBMの最初の会長兼CEOであるThomas J. Watsonにちなんで名付けられました。
